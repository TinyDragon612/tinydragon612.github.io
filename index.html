math is beautiful

One of my favorite visualizations: the Mandelbrot set.

If you'd rather use a local copy, save an image into images/mandelbrot.jpg and change the src to images/mandelbrot.jpg. For better performance, consider a WebP file.

Fun facts: the Mandelbrot set is the set of complex numbers c for which the sequence z_{n+1} = z_n^2 + c (starting at z_0 = 0) does not diverge. Tiny tweaks of parameters reveal intricate, self-similar patterns.
